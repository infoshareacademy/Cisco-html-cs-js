Porównanie zakresów zmiennych

Tabelka 1: Zakresy i widoczność

Słowo kluczowe Zakres Widoczna poza { }? Widoczna poza funkcją?
var Zakres funkcyjny (function scope) Tak Nie (jeśli zadeklarowana wewnątrz funkcji)
let Zakres blokowy (block scope) Nie Nie
const Zakres blokowy (block scope) Nie Nie

Tabelka 2: Podsumowanie typów zmiennych

Typ „Dom” zmiennej Przykład Widoczność
var Funkcja var x = 5; Widoczna w całej funkcji
let Blok { } let y = 5; Widoczna tylko wewnątrz bloku, w którym została zadeklarowana
const Blok { } const z = 5; Widoczna tylko wewnątrz bloku; wartość nie może być nadpisana

Uwaga: var ma zakres funkcyjny — oznacza to, że ignoruje zwykłe bloki { } (np. w if, for), ale jest ograniczona do funkcji, jeśli zadeklarowana wewnątrz funkcji.
